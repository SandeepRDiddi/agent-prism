Scene 5

Then move from prompt users to agent builders.

The training advances beyond chat interactions and teaches teams how to design agents, compose subagents, and map them to real enterprise data work.

01

Define the task

Pick a real workflow with repeatability, clear inputs, and measurable output.

02

Design the agent

Set instructions, tools, handoffs, controls, and success criteria.

03

Add subagents

Break complex tasks into focused workers for analysis, coding, review, or reporting.

04

Apply to data use cases

Examples include documentation, pipeline support, QA, metadata work, and research.

Sales takeaway: the program teaches capability creation, not just tool awareness.
